================================================================================
IMAGE DATA SOURCES
================================================================================

Backend Query: getPlayerInventory() → convex/products.ts
  Returns: product.image (URL string)
  Used by: Portfolio page

Backend Query: getProductBatchOrders() → convex/products.ts
  Returns: productImage (URL string)
  Used by: Company dashboard batch orders

Backend Query: getAllProducts() → convex/products.ts
  Returns: product.image (URL string)
  Used by: Marketplace page

================================================================================
IMAGE SPECIFICATIONS
================================================================================

Format:          <img src={imageUrl} alt={productName} />
Storage:         URL stored in product database
Type:            String URL
Required:        No (optional field with fallback)
Fallback Icons:
  - Marketplace Cart: Package icon
  - Portfolio: Shopping bag icon  
  - Dashboard Products: Package icon
  - Dashboard Orders: Package icon

CSS Styling:
  - border-radius: 0.5rem (rounded corners)
  - object-fit: cover (maintains aspect ratio)
  - object-position: center (centers image)

RESPONSIVE DESIGN
================================================================================

  - Portfolio: Card layout with medium images
  - Dashboard: Full table with visible images

Tablet (768px-1023px):
  - Marketplace: 2-column grid
  - Portfolio: Card layout maintained
  - Dashboard: Table with medium images

Mobile (< 768px):
  - Marketplace: 1-column grid, full width
  - Portfolio: Card layout, scrollable
  - Dashboard: Horizontal scroll or cards

================================================================================
PERFORMANCE METRICS
================================================================================

Image Sizes:
  - Marketplace (200px): ~100-300KB per image
  - Cart (80x80px): ~10-30KB per image
  - Portfolio (64x64px): ~8-20KB per image
  - Dashboard (48x48px): ~5-15KB per image

Total Page Impact:
  - Marketplace: +300-1000KB (5-10 products × images)
  - Portfolio: +80-200KB (10 products × images)
  - Dashboard: +50-150KB (10 products × images)

Load Strategy:
  - Images loaded on-demand (not preloaded)
  - No lazy loading (small thumbnails)
  - Browser native image caching

  → Browser handles gracefully
  → Shows alt text
  → Fallback icon visible

  → Browser retry mechanism
  → Page remains functional
  → No UI disruption
